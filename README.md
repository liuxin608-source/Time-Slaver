# 时间记录 · Time Tracker

一个极简、纯前端的时间记录工具。新建事件后开始计时，过程中可多次暂停/继续，结束时自动结算剔除暂停后的净时长，并按日/周/月汇总你把时间花在了哪里。

黑白灰极简风格，单文件、零依赖，所有数据保存在你浏览器本地（localStorage），不上传任何服务器。

## 功能

- **新建事件**：填事件名（可选）、选状态、选归属日期
- **七种预设状态** + 自定义文字：打工中 / 工作中 / 会议中 / 学习中 / 专注 / 休息 / 运动
- **计时**：全屏计时页显示状态与已持续时间，可任意多次暂停 / 继续
- **净时长**：自动剔除暂停时间；结束前显示「您已〔状态〕X 小时 X 分钟」
- **历史汇总**：按日 / 周 / 月 / 自定义区间查看，按状态统计总时长与占比
- **备份**：一键导出 / 导入 JSON，换设备或防止清缓存丢数据

## 使用

直接用浏览器打开 `index.html` 即可，无需安装、无需联网。

### 在线访问（GitHub Pages）

仓库 Settings → Pages → Source 选 `main` 分支、`/ (root)` 目录，保存后即可通过
`https://<你的用户名>.github.io/time-tracker/` 访问。

## 数据说明

记录保存在浏览器 `localStorage`，仅存在于你当前这台电脑的这个浏览器中。清除浏览器数据会丢失记录，建议定期用「导出备份」保存 JSON 文件。
